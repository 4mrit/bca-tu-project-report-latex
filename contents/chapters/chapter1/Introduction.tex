\section{Overview}


\para{
Online shopping has grown rapidly, and many small businesses also want to sell their products on the internet. However, building a full e-commerce website from scratch is expensive and requires technical skills. Because of this, a ready-made and customizable platform can help small sellers quickly launch their online store.

}
\para{
This project studies how an e-commerce system can be built using modern web technologies like Blazor and ASP.NET Core, and how it can be turned into a simple SaaS model where different shop owners can use the same platform.

}
