\section{Scope and Limitation}

\subsection*{Scope}
This project focuses on developing a basic e-commerce platform that allows small business owners to sell products online. The system includes the following features:
\begin{itemize}
  \item Product listing and product management
  \item Shopping cart and order placement
  \item Basic customer details handling
  \item Admin panel for managing the store
  \item Simple and user-friendly interface
  \item Web-based access so shop owners can use it from any device with a browser
\end{itemize}
The project demonstrates how a small online shop can be built and managed using modern web technologies.

\newpage
\subsection*{Limitations}
Although the system covers essential e-commerce functions, it has the following limitations:
\begin{itemize}
  \item It does not include advanced features such as multi-vendor support, analytics, or automation.
  \item Payment gateway integration is basic and may operate only in test mode.
  \item Security features such as full role-based access control or advanced encryption are limited due to time and academic scope.
  \item The system is not fully optimized for large-scale deployment and high traffic.
\end{itemize}
