\textbf{Brief Description:}
The diagram highlights the distribution of system components across nodes:
\begin{itemize}
    \item \textbf{Web Server Node:} Hosts the application server handling user requests.
    \item \textbf{Database Server Node:} Manages persistent storage for products, orders, and users.
    \item \textbf{Client Node:} Represents user devices (web browser or mobile app) accessing the system.
    \item \textbf{Optional External Services:} Payment gateway and notification services interacting with the system through secure interfaces.
\end{itemize}
