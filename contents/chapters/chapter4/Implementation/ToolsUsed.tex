\subsection{Tools Used}

\begin{itemize}
    \item \textbf{Programming Language:} C\# (.NET 8) -- Used for backend business logic, APIs, and server-side operations.
    \item \textbf{Web Framework:} Blazor Server -- Provides interactive web UI using C\# instead of JavaScript.
    \item \textbf{Database Platform:} PostgreSQL -- Relational database for storing product, user, and order information.
    \item \textbf{ORM:} Entity Framework Core -- Handles object-relational mapping and database interactions.
    \item \textbf{Authentication:} JWT (JSON Web Tokens) -- Implements secure, stateless authentication.
    \item \textbf{IDE/Development Environment:} Visual Studio 2022 -- Primary IDE for code development, debugging, and project management.
    \item \textbf{Version Control:} Git -- For source code management and collaboration.
    \item \textbf{Dependency Injection:} Microsoft.Extensions.DependencyInjection -- Facilitates modular and testable code.
    \item \textbf{Others:} 
    \begin{itemize}
        \item Bootstrap 5 -- Frontend styling.
        \item Swagger -- API documentation and testing.
        \item NuGet Packages -- For third-party dependencies (e.g., AutoMapper, FluentValidation).
    \end{itemize}
\end{itemize}
